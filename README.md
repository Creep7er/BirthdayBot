# BirthdayBot

Telegram-бот напоминает о днях рождения участников одного заданного группового чата. Все обновления из личных сообщений, каналов и других чатов игнорируются.

## Возможности

- добавление, обновление и удаление дней рождения администраторами;
- общий отсортированный список для участников чата;
- ежедневное поздравление с кликабельными упоминаниями;
- одно общее поздравление, если именинников несколько;
- защита от повторной отправки после перезапуска;
- хранение данных в SQLite на Docker volume.

## Требования

- Docker;
- Docker Compose;
- токен Telegram-бота и ID целевого группового чата.

## Настройка

Скопируйте `.env.example` в `.env` и заполните `BOT_TOKEN` и `ALLOWED_CHAT_ID`. Время проверки задают `TIMEZONE`, `BIRTHDAY_CHECK_HOUR` и `BIRTHDAY_CHECK_MINUTE`; путь базы — `DATABASE_PATH`.

## Запуск

```bash
docker compose up -d --build
```

## Команды

- `/addbirth 17.06` — ответом на сообщение пользователя, только для администраторов;
- `/addbirth @username 17.06` — только для администраторов;
- `/rmbirth` — ответом на сообщение, только для администраторов;
- `/rmbirth @username` — только для администраторов;
- `/birthdays` — список дней рождения;
- `/help` — краткая помощь.

Telegram Bot API не позволяет надёжно найти произвольного пользователя только по `@username`. Такой вариант работает для уже сохранённого пользователя либо когда Telegram предоставляет доступную боту информацию; иначе используйте команду ответом на сообщение.

SQLite-база хранится в named volume `birthday-data` по пути `/app/data/birthday_bot.sqlite3`. Логи смотрите командой:

```bash
docker compose logs -f birthday-bot
```

Дата `29.02` поддерживается: в високосный год поздравление отправляется 29 февраля, в невисокосный — 28 февраля.
